## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/hadishahpuri/space-shooter
cd space-shooter
```

### 2. Install Dependencies
```bash
go get .
```

### 3. Build the game
```bash
go build
```

Then simply run the game as an executable file like this:
```bash
./space-shooter
```
